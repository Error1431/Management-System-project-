# Student Management System

A comprehensive web-based Student Management System built with MERN stack (MongoDB, Express, React, Node.js).

## Features

- **User Authentication & Authorization**
  - Role-based access control (Admin, Teacher, Student)
  - JWT-based authentication
  
- **Student Management**
  - Add, edit, delete student records
  - View student profiles and statistics
  - Track enrollment and academic progress
  
- **Course Management**
  - Create and manage courses
  - Assign instructors
  - Track enrollment and capacity
  
- **Grade Management**
  - Record and calculate grades
  - Automatic GPA calculation
  - Grade reports and analytics
  
- **Attendance Tracking**
  - Mark daily attendance
  - View attendance reports
  - Calculate attendance percentage

- **Dashboard & Analytics**
  - Visual data representation
  - Real-time statistics
  - Performance metrics

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- Bcrypt for password hashing

### Frontend
- React 18
- React Router v6
- Axios for API calls
- Chart.js for data visualization
- Tailwind CSS for styling
- React Query for state management
